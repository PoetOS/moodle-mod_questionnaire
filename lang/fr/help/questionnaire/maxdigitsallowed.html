Question de type numérique : les paramètres "chiffres"

Utilisez Max. chiffres autorisés pour limiter le nombre de chiffres entrés pour une question de type numérique. Utilisez Nb de chiffres après la virgule pour forcer un certain nombre de chiffres après la virgule.
