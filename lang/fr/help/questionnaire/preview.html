Prévisualisation du questionnaire...

Le bouton Prévisualisation ouvre une fenêtre popup qui affiche le contenu de votre questionnaire tel que pourront le voir les étudiants.

Si le questionnaire comporte plus d'une seule page, toutes les pages seront affichées dans la fenêtre popup de prévisualisation, avec les sauts de page clairement indiqués.
