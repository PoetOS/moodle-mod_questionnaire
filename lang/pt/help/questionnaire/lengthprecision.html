Questão: Parâmetros Tamanho e Precisão

A tabela seguinte descreve o efeito destes dois parâmetros em cada tipo de questão.

Tipo Tamanho Precisão  
Sim/Não      
Caixa de Texto x caracteres de comprimento y caracteres no máximo Exibe uma caixa de texto, com x caracteres de comprimento na qual o utilizador não pode introduzir um texto com mais de y caracteres.
Caixa de Texto Multilinhas x colunas y linhas Exibe uma caixa de texto com x caracteres de comprimento e y linhas de altura (numero de linhas de texto).
Se não modificar o valor padrão 0 em x e y (ou se introduzir o valor 0), então o editor HTML será utilizado para editar o texto. Neste caso, as dimensões do editor serão as dimensões padrão no contexto da disciplina/utilizador.
Botões Rádio      
Caixas de Verificação x caixas no mínimo y caixas no máximo Use estes parâmetros para obrigar o utilizador a activar um mínimo de x e máximo de y caixas de verificação. Para obrigar a activar um número fixo de caixas de verificação, introduza o mesmo valor em x e y. Se apenas deseja um mínimo ou máximo, deixe o outro valor com o valor padrão 0. Se introduzir valores diferentes de 0 em x ou y, uma mensagem de aviso será exibida se o utilizador não cumprir os requisitos definidos. Neste caso, deve tornar estes requisitos claros para o utilizador explicando os mesmos nas instruções gerais do questionário ou no texto da questão.
Caixa de Combinação      
Classificação x itens y
  • x é o número de itens a usar na sua escala de classificação (Likert). Em geral é usado um valor entre 3 e 5). Se não alterar o valor padrão 0, então será assumido um valor de 5.
  • Se y for 1 será adicionada a coluna NS/NR após as colunas com a escala de classificação.
  • Se y for 2 serão impedidas respostas duplicadas em cada coluna.
    • Isto pode ser útil se pretende que o utilizador classifique um número de itens numa escala 1 a n sendo cada classificação única. Exemplo: Ordenar os itens A, B e C por ordem de preferência aceitará: A1, B3 e C2 ou A3, B2 e C1, mas não será possível seleccionar: A1, B1, C2 ou A1, B2, C2, etc.
    • Usado em conjunto com a opção escala personalizada, a opção 2 pode também ser útil se pretende que o utilizador estabeleça uma correspondência entre diversos itens, como por exemplo, as cores vermelho, azul, amarelo com o conjunto de estados mentais: entusiasmo, tranquilidade, concentração, onde cada cor apenas pode corresponder a um estado mental.
  • Se y for 3 será criada uma questão do tipo Diferencial semântico de Osgood. Este parâmetro deve ser usado em conjunto com a opção escala personalizada.
Data      
Numérica x tamanho y precisão

Use x para limitar o número de caracteres que podem ser introduzidos. Use y para formatar a resposta com y casas decimais. O número máximo de caracteres que o utilizador poderá introduzir será de x+y+1.

  • Exemplo 1: x=5; y=2. Resposta: 12 será automaticamente convertida para 12.00
  • Exemplo 2. x=5; y=2. Resposta: 12.569 erá automaticamente convertida para 12.57 (e uma mensagem de aviso será exibida ao utilizador informando da conversão)
  • Exemplo 3. x=5; y=2. Utilizador não poderá introduzir um mais longo que, por exemplo, 12.57898 (tamanho máximo = 5 (x) + 1 (separador decimal) + 2 (y precisão)= 8.
