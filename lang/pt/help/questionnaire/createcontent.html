Conteúdo inicial do Questionário

Pode criar um novo questionário de três formas alternativas. "Criar Novo"' é a forma por omissão:

  1. Criar Novo:
    Esta opção permite começar o questionário do zero. Seleccione esta opção e clique no botão "Continue"' para iniciar a criação do questionário.
  2. Copiar existente:
    Esta opção copia a estrutura de um questionário já existente para o questionário a criar. Pode copiar questionários do curso, ou de outros cursos, se os mesmos foram assinalados como "modelo". Pode pré visualizar os questionários disponíveis clicando nas respectivas designações. Active o botão próximo do questionário que pretende copiar e clique no botão "Continue" para editar o questionário.
  3. Usar Público:
    Esta opção atribui um questionário "público" existente a esta actividade. Active o botão próximo do questionário que pretende copiar e clique no botão "Continue". Neste caso a edição do questionário termina.
