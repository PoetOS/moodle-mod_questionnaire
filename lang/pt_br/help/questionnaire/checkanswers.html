Você deve inserir “!other” em uma linha para criar um campo opcional do tipo texto. Uma caixa !other usa como nome de campo Outro: como padrão, (ou a tradução equivalente para Outro no idioma que estiver em uso). Você pode alterar esse nome padrão utilizando o formato: “!other=nome do campo” como mostrado no examplo 1 acima.

Notas

  1. O tipo de questão check-box permite que o repondente marque várias respostas para uma questão.
  2. No momento, o tamanho do campo opcional de texto está definido para 25 caracteres e não é personalizável na interface online.
  3. Você pode utilizar mais que um campo opcional de texto para uma questão do tipo check-box, porém isso pode causar problemas e é extremamente não recomendado
  4. Quando um respondente responde a questão do tipo check-box que foi criada como Obrigatória e os parâmetros Número mínimo de marcações e Número máximo de marcações foram usados, uma mensagem de alerta será mostrada caso o respondente não marque o número necessário de caixas. Perceba que se o campo opcional de texto foi criado, se ele for marcado pelo respondente, ele contará como uma checkbox.
  5. Se o respondente marcar a cehckbox do campo opcional de texto e deixar o campo vazio, aquela checkbox será desmarcada(quando navegar para uma outra página ou submeter a enquete)
  6. If a respondent has cheked a checked a fill in the blank option check box and has filled in the text field, and later on changes his mind and unckecks that box, the fill in the blank text field will be automatically emptied (cool, ain't it?).
  7. Se um respondente marcar uma checkbox de um campo opcional de texto e preencher este campo, e só mais tarde mudar de idéia e desmarcar a caixa, o campo de texto será esvaziado automaticamente (divertido, não?)
