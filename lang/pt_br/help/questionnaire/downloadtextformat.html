Os tipo de questão botão de rádio/resposta única

Na página Fazer download em formato texto você tem duas opções que são exclusivamente sobre a saída das questões de botão de rádio/resposta única. O funcionamento destas opções é detalhado a seguir.

Texto da questão: Qual sua cor faorita? Opções: azul, verde, vermelho.

azul verde vermelho
Incluir código das opções 1 2 3
Incluir texto das opções azul verde vermelho
Incluir código e texto das opções 1 : azul 2 : verde 3 : vermelho
Não incluir caixa marcada = o mesmo que incluir texto da escolha da caixa maracada azul verde vermelho

Note que estas opções de saída são sobrescritas no caso de você ter selecionado a opção "escondida" quando criou suas questões de botão de rádio, por exemplo os valores nomeados serão exportados como dados. Veja este arquivo de ajuda.

O arquivo CSV

Se você precisar estudar os dados das respostas da enquete fora do Moodle, em um software como o Excel, o Open Office Calc ou outro pacote de software estatístico, você pode exportar os dados no formato de texto. O formato de texto disponível aqui é chamado CSV (valores separados por vírgula, do inglês comma separated values). Entretanto, para um uso mais universal, o separador tab (tabulação) é utilizado ao invés da vírgula ou do ponto-e-vírgula.

Quando você clica no botão Fazer download em formato texto, uma janela será aberta, perguntando se você gostaria de salvar o arquivo no seu computador ou abrí-lo diretamente. Selecione salvar. O nome do arquivo deve ser o nome da enquete atual, mas você pode alterá-lo. A extensão proposta para o arquivo é .txt, mas você também pode alterá-la. Note que você deve renomeá-la para .csv para poder importar o arquivo no Open Office Calc.

Agora vá ao software que você deseja usar e abra o arquivo. As próximas operações dependem do software que você está usando e de uma série de outros parâmetros. Se você está utilizando um pacote de idiomas não ASCII no Moodle, ou se você está usando o idioma inglês mas algumas das perguntas (ou respostas) da enquete possuem caracteres não ASCII, então o software que você usa deve ser capaz de detectar automaticamente que o seu arquivo está codificado em UTF-8 e decodificá-lo de acordo. Por favor, note que se apenas uma pequena parte dos caracteres (ou nenhum deles) forem não ASCII, a detecção automática pode não funcionar e você terá que selecionar a codificação em UTF-8 manualmente.

No topo da "tabela" importada, a primeira linha contém o nome de todos os campos da questão. Isto é seguido por uma resposta por linha. Os primeiros sete campos são:

Resposta Submetida em: Instituição Departmento Curso Grupo ID Nome completo Nome de usuário

O campo submitted_date contém a data (e o horário, i.e. hora minutos e segundos) na qual a resposta foi submetida pelo respondente. Por exemplo: 15/09/2007 15:57. Normalmente você deve poder usar a formatação de células para mostrar datas no formato desejado, nas convenções de formatação do seu idioma, incluindo segundos e, se você desejar, o nome completo dos dias ou meses, mantendo a facilidade para ordenação.

Se a enquete foi configurada para "respostas anônimas", o ID e o "Nome de usuário" estão vazios e o campo "nome completo" mostrará "Anonymous".
