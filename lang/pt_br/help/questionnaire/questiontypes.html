Tipos de Questões

Selecione o tipo de questão pela lista suspensa e clique no botão Adicionar tipo de questão selecionado. Esta opção é o que definirá como cada questão se parecerá.

Você pode obter mais ajuda para cada tipo de questão na tela de edição das mesmas.

---Quebra de Página---

Use esta opção para dividir a enquete em mais de uma página, para o caso de enquetes muito grandes.

Atenção, se a página contiver questões em que a resposta seja obrigatória, o usuário não poderá navegar para a próxima página enquanto não responder as mesmas.

Data
Use o formato dia/mes/ano ex. para dia 14 de Março de 1945: 14/3/1945

Use este tipo de questão se você espera que a resposta seja uma data formatada corretamente.

O formato vai depender do idioma utilizado pelo usuário, por exemplo:

  • 4/21/2007 (Inglês/Estados Unidos)
  • 21/4/2007 (Inglês/Reuno Unido)
  • 21-4-2007 (Francês/França)

Não se preocupe, um exemplo será exibido no questionário.

Se for informado um um formato de data "impossivel" ou formatado de maneira incorreta, ele será reformatado corretamente (se for possível) ou uma mensagem de erro vai ser exibida ao usuário.

Atenção: Para que as datas possam ser exportadas corretamente para uma planilha eletrônica, a data digitada pelo usuário precisa estar entre 1902 e 2037. Se a data esperada estiver fora desse limite, então utilize uma questão do tipo Caixa de texto.

Observação: As questões do tipo data, aceitam também data que consistam somente do mês junto com o ano (ex: 12/2008 para dezembro de 2008) ou somente o ano (ex: 2008). Porém, para compatibilidade com as planilhas eletrônicas, essas datas incompletas serão automaticamente transformadas em datas completas no formato dd/mm/yy, ex: 12/2008 -> 01/12/2008 e 2008 -> 01/01/2008.
Talvez você precise explicar essa peculiaridade aos seus alunos antes dele utilizarem a enquete.

Lista suspensa

Não existe nenhuma vantagem em usar uma Lista suspensa no lugar de alguns Botões de opção exceto no caso de existir muitas opções, evitando assim ocupar muito espaço da tela.

Área de texto

Algumas configurações estão disponíveis para permitir ativar ou desativar o editor HTML e para personalizar o tamanho e largura da área de texto.

Rótulo
Iste não é um tipo de questão, mas sim um (pequeno) texto que será exibido para auxiliar na explicação sobre uma série de questões.

Campo numérico

Use este tipo de questão se você espera que a resposta seja um número corretamente formatado.

Você pode especificar a quantidade de digitos e o número de casas decimentais requeridas, utilizando os parâmetros:

  • Quantidade máxima de digitos permitidos
  • Número de casas decimais
