Check Boxes Question parameters

Use these parameters to force respondent to tick a minimum of Min. boxes and a maximum of Max. check boxes. To force an exact number of check boxes to be ticked, set Min. and Max. to the same value. If only a min or a max value is desired, just leave the other value to its default 0 value. If you set Min. or Max. to values other than their default 0 value, a warning message will be displayed if respondent does not comply with your requirements. Obviously you should make any requirements clear to the respondent either in the general instructions of your Questionnaire or in the text of relevant questions.
