Apertura y cierre de un cuestionario

Puede especificar la fecha de cierra de un cuestionario aquí. Para ello, active el recuadro y seleccione la fecha y la hora que desee.

Una vez que se cumpla el plazo estipulado, los usuarioas no podrán responder el cuestionario. Si no se activa el recuadro, estará disponible permanentemente.
